# Changelog

## [1.0.1] - 2025-11-25

### Added
- 一括ディスクリプション生成＆登録機能（サブメニュー）
- 記事一覧から複数記事を選択してディスクリプションを一括生成
- 生成したディスクリプションを抜粋フィールドに一括登録
- プラグイン一覧に「設定」「一括生成」リンクを追加
- 投稿タイプ・Desc状態・抜粋状態でのフィルタリング機能
- ソート機能（ID、タイトル、日付、抜粋、ディスクリプション）

## [1.0.0] - 2025-09-10

### Added
- 初回リリース - OpenAI GPTによるメタディスクリプション自動生成機能
- OpenAI GPT-4.1 Nano、Mini、標準モデル対応
- 投稿・固定ページ・カスタム投稿タイプ・メディアファイル対応
- 管理画面での詳細設定機能（モデル選択、文字数設定、投稿タイプ選択）
- OpenAI APIキーテスト機能
- 生成されたディスクリプションのワンクリックコピー機能
- WordPress管理画面に統合されたユーザーフレンドリーなインターフェース
- 投稿タイプ別の有効化制御機能
- 80文字から500文字まで対応した文字数設定
- セキュアなAJAX処理とnonce検証
- メディアファイルの詳細情報（ファイル名、代替テキスト、MIME type）を考慮した生成
